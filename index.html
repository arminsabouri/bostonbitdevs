Boston BitDevs is a Bitcoin-focused, technical meetup. Like all worldwide BitDevs groups, we run a Socratic Seminar (learn more here).

If you're looking to learn about Bitcoin's latest technical developments, this is the event for you! Whether you're a long-time Bitcoiner looking to participate in the discussion, or a new Bitcoiner hoping to soak up technical information like a sponge, all are welcome.

The reading lists for Socratic Seminars 1-8 can be found in the past events section of the Meetup page.
